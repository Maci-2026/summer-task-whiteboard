# 暑假周任务拖拽白板

这是一个纯前端 H5/PWA 版本，可部署到 GitHub Pages、Cloudflare Pages 或腾讯 CloudBase 静态网站托管。

## 当前能力

- 两个孩子切换
- 周一到周日、上午/下午/晚上投放棋盘
- 右侧任务图标库
- 顶部常用任务快速拖拽栏
- 拖拽投放；手机也可先点格子再点任务票
- 固定事项、自由时间、周末复盘
- 本机浏览器自动保存
- JSON 导入/导出
- PWA manifest 和 service worker，可添加到 iPad/iPhone 主屏幕

## 免费部署建议

### GitHub Pages

优点：免费、已经可用、部署简单。
缺点：中国大陆网络和微信内打开速度可能不稳定。

### 腾讯 CloudBase 静态网站托管

优点：国内访问和微信里打开更顺。
缺点：需要腾讯云账号、创建免费体验环境，免费额度和续期规则以腾讯云为准。

部署时上传本文件夹内全部文件即可：

- `index.html`
- `manifest.json`
- `sw.js`
- `assets/`

## 后续同步版

当前版本数据保存在本机浏览器。如果要让家长和孩子多设备实时同步，需要接入云数据库，例如：

- 腾讯 CloudBase 数据库
- Supabase
- Firebase

推荐下一步接腾讯 CloudBase 数据库，这样微信和国内网络更稳定。
